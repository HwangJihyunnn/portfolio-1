사용자가 홈페이지를 신뢰할 수 있는 사이트를 만들어 가며,

불필요한 태그가 아닌, 의미있는 시맨틱 태그가 되어 같이 협업하여 함께 성장하고 싶습니다.

자기개발에 끝이 없는 것처럼 끝없이 성장하고 개발하겠습니다.

자기주도+습득력+센스 = "황지현"

Skill

  • 유효성 문제없이 HTML5의 웹표준과 웹접근성을 준수하고 크로스 브라우징이 가능합니다.
  • CSS3로 다양한 최신 트렌드 디자인과 애니메이션을 구성할 수 있습니다.
  • 슬라이드, 메뉴, 탭메뉴 애니메이션 기능 등 최신 트렌드에 적용된 다양한 동적 기능을 자바스크립트로 구현하며 활용도 가능합니다.
  • git을 이용하여 프로젝트 관리가 가능합니다.
  • 부트스트랩을 이용하여 빠르게 디자인을 구축할 수 있습니다.
  • sass를 이용하여 관리를 편리하게 할 수 있습니다.
  • 리액트는 계속 배움중에 있으며 현재는 간단한 html문서를 껴넣을 수 있습니다.
  • 이메일 로그인 / 글쓰기 / 글삭제 등을 관리할 수 있습니다.

OAK VALLEY ㅣ 오크밸리

  • 페이지설계 : 반응형 Main + Sub*3(소개/다이닝/객실)

  • 제작기간 : 40시간 퍼블리싱 100%

  • 웹 크기 : max-width: 1800px

    태블릿 크기 : max-width: 1024px

  • 프로젝트 개요 :

    오크밸리 사이트를 클론 코딩하면서 자바스크립트와 제이쿼리라이브러리를 활용한 포트폴리오 사이트를 구현해보았습니다. 메인의 슬라이드 기능은 플러그인을 사용하지않고 직접 구현하였으며, 스크립트로 구현할 수 있는 부분은 최대한 직접 만들어보았습니다.

Center ㅣ 세상의 중심, 센터

  • 페이지설계 : 모바일 Main + Sub*3 + 카카오로그인 + 글쓰기 및 삭제 수정

  • 제작기간 : 5일 4명 / 퍼블리싱25%

  • 모바일 크기 : max-width: 430px

  • 프로젝트 개요 :

    파이어베이스를 활용하여 이메일 로그인 구현과 게시글 삭제 및 수정 업로드를 하였으며 또한 카카오 api를 이용하여 카카오 로그인까지 구현하였습니다. 익숙치 않은 플러그인이였지만, 미처 몰랐던 부분이나 놓쳤던 부분도 챙기고 배우면서 기능적인 부분뿐만 아니라 의사소통 하는 법을 배우고 조금 더 성장하는 계기가 되었습니다.

    본 화면은 모바일 430px크기에 최적화 되어있습니다.
